## Features

- **15+ Components** — Button, Card, Modal, Dropdown, and more
- **Design Tokens** — OKLCH colors with automatic dark mode
- **Layout Primitives** — Stack, Row, Grid, Center utilities
- **TypeScript** — Full type safety out of the box
- **Lightweight** — No runtime dependencies beyond SolidJS
