## Layout Primitives

### Stack

Vertical layout with consistent spacing.

```html
<div class="stack">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```

| Class | Gap |
|-------|-----|
| `.stack` | `--space-md` (0.75rem) |
| `.stack-sm` | `--space-sm` (0.35rem) |
| `.stack-lg` | `--space-lg` (1.5rem) |

### Row

Horizontal layout with alignment.

```html
<div class="row">
  <span>Left</span>
  <span>Right</span>
</div>
```

| Class | Description |
|-------|-------------|
| `.row` | Horizontal flex with center alignment |
| `.row-sm` | Smaller gap (`--space-xs`) |
| `.row-lg` | Larger gap (`--space-md`) |
| `.row-between` | Space between items |
| `.row-end` | Align to end |
| `.row-start` | Align items to start |

### Grid

Responsive auto-fit grid.

```html
<div class="grid">
  <div>Card 1</div>
  <div>Card 2</div>
  <div>Card 3</div>
</div>
```

| Class | Columns |
|-------|---------|
| `.grid` | Auto-fit (min 250px) |
| `.grid-2` | 2 columns |
| `.grid-3` | 3 columns |
| `.grid-4` | 4 columns |

Customize the minimum column width:

```html
<div class="grid" style="--grid-min: 150px">...</div>
```

### Center

Max-width centered content.

```html
<div class="center">
  Centered content with max-width
</div>
```

| Class | Max Width |
|-------|-----------|
| `.center` | 65ch |
| `.center-narrow` | 45ch |
| `.center-wide` | 90ch |

### Cluster

Wrapping horizontal layout for tags/buttons.

```html
<div class="cluster">
  <span class="badge">Tag 1</span>
  <span class="badge">Tag 2</span>
</div>
```

### List

Styled lists with proper markers.

```html
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
```

Works with both `<ul>` and `<ol>`, with automatic nested styling.

| Class | Description |
|-------|-------------|
| `.interactive` | Border highlights on hover |

| Class | Description |
|-------|-------------|
| `.hidden` | Display none |
| `.sr-only` | Screen reader only |
| `.hover-reveal` | Show on parent hover |
| `.hide-mobile` | Hide on mobile (< 768px) |

| Class | Description |
|-------|-------------|

| `.animate-pulse` | Pulsing animation |
