---
title: Design Tokens
description: CSS custom properties for theming
---

## Colors

### Backgrounds
| Token | Description |
|-------|-------------|
| `--bg` | Primary background |
| `--bg-alt` | Alternate/subtle background |
| `--border` | Border color |

### Text
| Token | Description |
|-------|-------------|
| `--fg` | Primary text |
| `--fg-muted` | Secondary text |
| `--fg-subtle` | Tertiary text |
| `--fg-faint` | Faint/disabled text |

### Accent
| Token | Description |
|-------|-------------|
| `--accent` | Primary accent color |
| `--accent-fg` | Text on accent background |

### Status
| Token | Description |
|-------|-------------|
| `--success` / `--success-fg` | Success state |
| `--error` / `--error-fg` | Error state |
| `--warning` / `--warning-fg` | Warning state |

## Typography

| Token | Value |
|-------|-------|
| `--font` | System sans-serif stack |
| `--font-mono` | System monospace stack |
| `--text-sm` | 0.875rem |
| `--text-base` | 1rem |
| `--text-lg` | 1.25rem |
| `--text-xl` | 1.5rem |

## Spacing

| Token | Value |
|-------|-------|
| `--space-xs` | 0.25rem |
| `--space-sm` | 0.5rem |
| `--space-md` | 1rem |
| `--space-lg` | 1.5rem |
| `--space-xl` | 2.5rem |

## Other

| Token | Value |
|-------|-------|
| `--radius` | 0.25rem |
| `--radius-lg` | 0.5rem |
| `--shadow` | Subtle shadow |
| `--shadow-lg` | Larger shadow |
| `--transition` | 150ms ease |
